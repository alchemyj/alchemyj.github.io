    Community edition

    Introduction

    AlchemyJ community edition is the free version you can download from alchemyj.io. It does not require an AlchemyJ license to run.

    What are the differences of the community edition as compared to the enterprise edition?

    • Does not require a license to run a generated project.
    • Supports only one REST API endpoint group per package.
    • Supports at most 4 function points per package.
    • Supports debug mode and standard mode only. No high performance mode support.
    • Supports MySql only. No Oracle and MS SQL support.
    • Supports Windows workstation and Windows Server only. This means that the generated project will not run on non-Windows OS such as Linux.
    • REST API generated by community edition will have a short ad message in all API responses such as the one below:
    {
    "Output": "Hello TIMMY, now is 15:59:03",
    "_alchemyj_info": {
        "statusCode": 200,
        "responseDatetime": "23/01/2020 15:59:07",
        "_link": "/MyAPI/SayHello"
        "Edition": "This API was generated by AlchemyJ community edition. Download AlchemyJ from http://www.alchemyj.io  to start building your own API."
}
    • No JS Data Dictionary.
