    set-formula-to-cell

    Description

    Sets a formula to a specified cell.

    Supporting Parameters

    Parameter NameTypeMandatoryDefault ValueDescription
    functionPointStringYesNoneDefines which function point this action would be triggered, value “*” means the action would be triggered by all function points.
    SettersMapNoNoneThe key of the Map is the cell address. The value is the formula to be set.

    Sample

    [  
  {  
       "action" : "set-formula-to-cell",  
       "parameters" : {  
           "functionPoint" : "*",  
           "setters" : {  
               "sheetA!A1" : "sum(A2, A3)",  
               "sheetA!B1" : "now()"
          }  
      }  
  }  
]
