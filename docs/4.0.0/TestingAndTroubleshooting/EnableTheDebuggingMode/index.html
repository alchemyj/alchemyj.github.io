    Enable the Debugging Mode

    AlchemyJ provides a debugging mode for the developer to test and debug their program by using the Debug Console. You can enable it by doing the following steps.

    1. Go to the AlchemyJ toolbar, click the Properties.

    2. Move to the Build tab on the dialog box. Select Debug on the Compilation mode.

    3. Move to the Debug tab, to specify the Debugger console port, which would be used by the Debug Console. It's 9898 by default. Please make sure the port is not used by other programs.

    4. Define the file location Debugger Command Action File.

    5. Build and run your program.

    6. Open Debugger Command Action File to specify the debug command actions you want to run.

    7. Open Debug Console from the AlchemyJ toolbar and click launch debug console.

    8. Run an API function in REST or in your Java project. Debug messages shall appear in the Debug Console.
