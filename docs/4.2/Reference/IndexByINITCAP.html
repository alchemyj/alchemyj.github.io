    Function NameFunction Description
    ajAddressReturns a cell reference or a range reference as text.

    B

    Function NameFunction Description
    ajBigNumAbsReturns a BigNumber whose value is the absolute value, i.e. the magnitude, of the value of this BigNumber.
    ajBigNumCompareToCompares if BigNumber number1 is greater than number2. Return 1 if number1 is greater than number2.
    ajBigNumDivideReturns a BigNumber whose value is the value of this BigNumber number1 divided by number2.
    ajBigNumDivideToIntegerReturns a BigNumber whose value is the integer part of dividing the value of this BigNumber number1 by number2.
    ajBigNumEqualsReturns true if the value of this BigNumber number1 is equal to the value of number2, otherwise returns false
    ajBigNumIntReturns a BigNumber whose value is the value of this BigNumber rounded to an integer using rounding mode round_mode.
    ajBigNumIsGreaterReturns true if the value of this BigNumber number1 is greater than the value of number2, otherwise returns false.
    ajBigNumIsGreaterEqualReturns true if the value of this BigNumber number1 is greater than or equal to the value of number2, otherwise returns false.
    ajBigNumIsIntegerReturns true if the value of this BigNumber is an integer, otherwise returns false.
    ajBigNumIsLessReturns true if the value of this BigNumber number1 is less than the value of number2, otherwise returns false.
    ajBigNumIsLessEqualReturns true if the value of this BigNumber number1 is less than or equal to the value of number2, otherwise returns false.
    ajBigNumIsNANReturns true if the value of this BigNumber is NaN, otherwise returns false.
    ajBigNumIsNegativeReturns true if the sign of this BigNumber is negative, otherwise returns false.
    ajBigNumIsPositiveReturns true if the sign of this BigNumber is positive, otherwise returns false.
    ajBigNumIsZeroReturns true if the value of this BigNumber is zero or minus zero, otherwise returns false.
    ajBigNumMaxReturns a BigNumber whose value is the maximum of the arguments.
    ajBigNumMinReturns a BigNumber whose value is the minimum of the arguments.
    ajBigNumModReturns a BigNumber whose value is the value of this BigNumber number1 modulo number2, i.e. the integer remainder of dividing this BigNumber by number2.
    ajBigNumMultiplyReturns a BigNumber whose value is the value of this BigNumber number1 multiplied by number2.
    ajBigNumNegateReturns a BigNumber whose value is the value of this BigNumber negated, i.e. multiplied by -1.
    ajBigNumNumberValueReturns the value of this BigNumber as a JavaScript number primitive.
    ajBigNumPowerReturns a BigNumber whose value is the value of this BigNumber number1 exponentiated by number2.
    ajBigNumPrecisionReturns a BigNumber whose value is the value of this BigNumber rounded to a precision defined in trailing_zeros.
    ajBigNumRandReturns a new BigNumber with a pseudo-random value equal to or greater than 0 and less than 1.
    ajBigNumRoundReturns the value of this BigNumber after rounded.
    ajBigNumShiftReturns a BigNumber whose value is the value of this BigNumber number1 shifted by the no. of places defined in number2.
    ajBigNumSqrtReturns a BigNumber whose value is the square root of the value of this BigNumber.
    ajBigNumSubtractReturns a BigNumber whose value is the value of this BigNumber number1 minus number2.
    ajBigNumSumReturns a BigNumber whose value is the value of this BigNumber number1 plus number2.
    ajBigNumSumRangeReturns a BigNumber whose value is the sum of the arguments.
    ajBigStrLeftReturns the specified number of characters from the start of BigString text.
    ajBigStrLowerConverts all characters in BigString text to lowercase.
    ajBigStrMidReturns the characters from the middle of a BigString text, given a starting position and length.
    ajBigStrProperConverts the BigString text to proper case; the first letter in each word is uppercase, and all other letters to lowercase.
    ajBigStrReplaceReplaces part of a BigString text with a different text string.
    ajBigStrReptRepeats BigString text a given number of times.
    ajBigStrRightReturns the specified number of characters from the end of BigString text.
    ajBigStrSetSaves plain text to memory and returns the memory id.
    ajBigStrSubstituteReplaces existing text with new text in a BigString.
    ajBigStrTextJoinConcatenates a list or range of BigString text using a delimiter.
    ajBigStrTrimRemoves all spaces from a BigString text except for single spaces between words.
    ajBigStrUpperConverts a BigString text to all uppercase letters.
    ajBigStrValueShow the value of passed memory ID.
