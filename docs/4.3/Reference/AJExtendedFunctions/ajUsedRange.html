    ajUsedRange function

    Description

    The ajUsedRange function returns the used Excel range of the specified cell reference. This function is only used for defining worksheets in the Related Worksheet section in the REST API Endpoint Group and Java API class. It should not be used in any business logic worksheet since there is no corresponding Java implementation for it.

    Syntax

    ajUsedRange(cell_reference)

    Argument NameArgument TypeDescription
    cell_reference

    (required)    		RangeA cell range of cells and obtain the used address of the cell range.

    The function will return:

    1) Return Value: Address of the Excel range

    2) Return Type: Single Value / Multiple values (array formula)

    Example

    =ajUsedRange(Sheet1!A1:Z1)

    If the worksheet exists, a value such as Sheet1!B3:K4 would be returned. This means that the top left cell that is used in the Sheet1is B3 and the bottom right cell that is used is K4.

    Error Scenarios

    It will return #VALUE! when missing any required parameter or mismatch parameter type. Besides, system will raise error for below scenario(s).

    Error Scenario
    Referring range does not exist.
