Write your first Java API

This document will guide you through the process of setting up a Java API using AlchemyJ. At the end, you will be able to build a Java API which retrieves customer data from a data without writing any Java code.

Before we start

As this project relies on data in a table called tb_customer. Create the below table in your database first.

Create a AlchemyJ Workbook

  1. Launch Microsoft Excel and start with a new blank workbook. Save as Customer Management Service.xlsm. Please ensure that you select Excel Macro-Enabled Worksheet as the file type.

  2. Go to AlchemyJ tab, click on "New Project".

  1. Three system worksheets (%%SysRuntime, %%SysConfig, and %%AppConfig) should be created and initialized.

  1. In System Configuration Worksheet, below configurations need to be modified to adapt to your local environment. Properties not defined below could be left as default or you can modify them according to your need. Refer to SysConfig Worksheet for details of each configuration.

Define a Business Logic Sheet

  1. Create a new worksheet and name it GetCustomer. We will put the formulas for the Get Customer function point on this worksheet.
  2. User can use this function to retrieve a customer record from the database by providing a customer name. We expect the input value will be set in cell C12.
  3. Use the input field C12 to concatenate the SQL statement to retrieve the record from database table which will be used in ExecuteSql .
="SELECT CUSTOMER_NAME, DOB, GENDER, CUSTOMER_LVL, TELEPHONE FROM TB_CUSTOMER WHERE CUSTOMER_NAME ='" & C12 &"'";
  1. Select B4:F4 and input following formula just as in B7. Press CTRL+ SHIFT+ ENTER after you finished inputting the formula to make it become an Array Formula.
=IF(AlchemyJ_FunctionPoint = "Get Customer", ExecuteSql(B10), "")

Preview and Compile Your Program

  1. Now all formula and settings have been defined. We shall preview the API in AlchemyJ Studio. Go to AlchemyJ toolbar, click Run Function Point and select Get Customer to test the Get Customer function point in AlchemyJ Studio.

The result can be found in B4:F4. The model works as expected.

  1. After we have test the API in AlchemyJ Studio, it is time to compile the model into Java code. Go to AlchemyJ toolbar, click Build Project.

    If you want AlchemyJ to generate the Jar package, check the Compile the project into a package checkbox. Otherwise, leave it unchecked. AlchemyJ will generate the Java project only.

    Click Build to start building the project. After AlchemyJ validates and analyse the workbook, AlchemyJ Compiler will start compiling the workbook.

  1. When compilation finishes, you shall see a java project has been generated. The default output folder is the Build folder in the folder that the workbook is in. As shown in the below screen shot, a java class called CustomerService has been created. Now you have got the Get Customer API in Java Language!

  1. To use this generated package, you need import the package to your existing Java Project. Refer to Using AlchemyJ Java API in a Java project to learn how to use the generated Java package in a Java project.
