ReadFile function

Description

The ReadFile function imports the content of a file into a defined area in Excel worksheet. It is mainly used for loading data from a text file to Excel for processing.

Syntax

ReadFile(File_location, Column_separator, [Left_enclosure_character], [Right_enclosure_character], [Begin_at_line], [Number_of_lines], [Run_by_function_point_only])

Argument Name Description

File_location

(required)

 To specify the path of the file to be read.

Column_separator

(required)

To specify that character that is used to split input record into fields. In the case of a CSV file, comma is usually used.

If the file does not contain any column separator, please provide an empty string (“”).

Left_enclosure_character

(optional)

To specify the left enclosure character. When a Column_separator is used, a field value itself can actually contain a column separator character. For example, when using a comma delimited format and with a field containing "Cathy, Chow" as its actual value, the field would be separated into two fields as there is a comma in. Using an enclosure character in write and read operation can solve this problem.

If you do not specify anything, the default value will always be an empty string (“”) which means no enclosure character.

Right_enclosure_character

(optional)

To specify the right enclosure character.

If you do not specify anything, the default value will always be an empty string (“”) which means no enclosure character.

Begin_at_line

(optional)

To specify the starting line to begin the extraction of the content of the file.

If you do not specify anything, it will start from the beginning.

Number_of_lines

(optional)

To specify the number of lines to be extracted from the file.

If you do not specify anything, it will read all the lines starting from Begin_at_line and till the end of file.

Run_by_function_point_only

(optional)

If it equals FALSE, the function can be executed through ‘Excel Calculation’ (can be either ‘Automatic’ or ‘Manual’; ‘Calculate Now’ or ‘Calculate Sheet’) or ‘Run Function Point’.

If it equals TRUE, the function can be executed with ‘Run Function Point’ (AlchemyJ toolbar -> Run Function Point) only.

If you do not specify anything, the default value will always be TRUE.

The function will return:

1) Content type: contents of the selected file

2) Method: cell array
